# MyFirstApp

This is a React Native app built using Expo.  
The app displays a user profile with a profile picture, user name, and a button that toggles a greeting message.
The profile provided is a demo profile.

This project demonstrates the use of:
- Components
- Props
- State
- Styling with `StyleSheet.create()`
- User interaction (button click)
